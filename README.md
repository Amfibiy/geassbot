# Geass Collector Bot 🤖

Бот для организации 30-минутных сборов участников в группах Telegram с глубокой статистикой.

## Стек технологий
- **Язык:** Python 3.10+
- **Библиотека:** pyTelegramBotAPI
- **База данных:** MongoDB (Atlas)
- **Деплой:** Render / Docker

## Основные функции
- 🚀 **Сборы:** Автоматические 30-минутные циклы с обновлением счетчика.
- 👤 **База данных:** Авто-регистрация участников и ручное добавление по @username.
- 📊 **Статистика:** Просмотр истории сборов за день, неделю, месяц или произвольный период.
- 🧹 **Очистка:** Гибкое управление историей данных.
- 🔒 **Доступ:** Разделение прав (Админ / Участник) и контекстные меню команд.

## Установка и запуск
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Создайте файл `.env` и добавьте:
   - `BOT_TOKEN=ваш_токен`
   - `MONGO_URI=ваша_ссылка_на_mongo`
4. Запустите: `python main.py`
