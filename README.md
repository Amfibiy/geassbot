# Geass Collector Bot 🤖

Бот для организации и координации сборов участников в групповых чатах Telegram с гибкими настройками и статистикой.

## Основные функции
- 🚀 **Сборы:** Автоматические циклы сбора с настраиваемой длительностью.
- ⚙️ **Гибкость:** Настройка времени сбора (от 1 мин) и часового пояса (дефолт: МСК+2) через `/settings`.
- 📊 **Статистика:** Просмотр истории сборов за любой период (день, неделя, месяц).
- 🧹 **Очистка:** Возможность удаления устаревших записей из базы данных.
- 🔒 **Доступ:** Настройки доступны только администраторам в приватном чате с ботом.

## Команды
- `/start` — Приветствие и краткая инструкция.
- `/settings` — Настройка времени сбора и часового пояса (только в ЛС).
- `/collect` — Запуск основного сбора в группе.
- `/test` — Запуск тестового сбора без сохранения в базу.
- `/list` — Просмотр истории участников и сборов.
- `/stop` — Остановка активного сбора.

## Технологии
- **Язык:** Python 3.10+ (pyTelegramBotAPI)
- **База данных:** MongoDB
